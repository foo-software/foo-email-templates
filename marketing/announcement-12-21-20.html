Hello, we've made changes to our service that may have impacted your Lighthouse scores as they appear on Foo. On 12/12/2020 we changed the way we run Lighthouse to instead utilize PageSpeed Insights API vs running it directly. With this change we now run Lighthouse from the same API as web.dev and PageSpeed Insights (both owned by Google).

We are sorry if this change had a negative impact on your Ligthhouse scores, but we believe it was necessary to help establish parity with Google when possible which was a main driving factor of this change. We wanted to announce this change when deemed stable, however we realize it would have been better to over-communicate by announcing before the actual change... lesson learned.

You can read more about the motivation behind this change here.
