In light of Google's recent announcement of its algorithm change in May, we've revamped and revived our Web Vitals product!

Web Vitals

When Google announced its algorithm change to prioritize page experience, it recommended utilizing Web Vitals to analyze field data. Web Vitals is an open source project to gather page experience data. Foo provides a service for storage of Web Vitals data and a GUI to visualize and analyze. Read more about Web Vitals and our new product here!

Web Vitals Dashboard or Data Studio

Getting started with Foo's Web Vitals Implementation is as simple as adding a JavaScript snippet to your website. After you've setup the snippet, Foo will collect Web Vitals data in real time and provide charts when logged in via our dashboard. We also have a Data Studio Connector... check it out!

Lighthouse with CrUX

To prepare for the new search update, Lighthouse still remains a great tool for lab data. When building or maintaining a website, it's the industry standard for analyzing website performance and other page experience characteristics, providing feedback from an engineering perspective. We've added CrUX data in addition to Lighthouse reporting. If you're not utilizing our Lighthouse monitoring solution, now is a better time than ever!
